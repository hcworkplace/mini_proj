# 기사본문으로 헤드라인 뽑기
### 1. 데이터 준비(크롤링)
* 네이버 뉴스 사회면 크롤링
- - -
### 2. 데이터 전처리
- 기사 내용이 너무 짧거나 없는 경우 제거(예: 사진만 있는 경우)
- 기사 본문에서 내용 아닌 것 제거(기자 이름, 이메일 ....)
- 기사 제목 중 제목이 아닌 부분 제거([속보], (종합),(상보),…)
- 한문제목 가지는 기사 데이터에서 제거
---
- Mecab.moprhs 를 이용해  토큰화
- 불용어 제거 (의, 가, 이, 은, 들, 는, 좀, 인데요, 다, 을, 습니다 …)
- 정수인코딩
- 최대길이로 패딩 / 트레인-테스트 셋(3:1) 분리
